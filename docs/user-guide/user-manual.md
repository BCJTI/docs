## Getting Started

### Logging In
1. Navigate to your SmartComex login page
2. Enter your username and password
3. Complete two-factor authentication if enabled
4. Click "Sign In" to access your dashboard

### First-Time Setup
After logging in for the first time:
1. Complete your business profile
2. Set up payment methods
3. Configure shipping options
4. Add your first products

## Product Management

### Adding Products

1. **Navigate to Products**
   - Click "Products" in the main navigation
   - Select "Add New Product"

2. **Basic Information**
   - Product name and description
   - SKU (Stock Keeping Unit)
   - Category and tags
   - Product images

3. **Pricing**
   - Base price
   - Sale price (optional)
   - Tax settings
   - Currency options

4. **Inventory**
   - Stock quantity
   - Inventory tracking settings
   - Low stock alerts
   - Backorder options

5. **Shipping**
   - Weight and dimensions
   - Shipping class
   - Special shipping requirements

### Managing Product Variants

For products with multiple options (size, color, etc.):

1. **Enable Variants**
   - Toggle "This product has variants"
   - Define variant attributes (size, color, style)

2. **Create Variants**
   - Generate all possible combinations
   - Set individual pricing and SKUs
   - Manage inventory per variant

3. **Variant Display**
   - Configure how variants appear to customers
   - Set default variant selection
   - Upload variant-specific images

### Product Categories

Organize your products with a hierarchical category system:

1. **Create Categories**
   - Navigate to Products → Categories
   - Add main categories and subcategories
   - Set category descriptions and images

2. **Assign Products**
   - Assign products to appropriate categories
   - Products can belong to multiple categories
   - Use bulk actions for multiple assignments

## Order Management

### Order Processing Workflow

1. **New Orders**
   - Orders appear in the "New" status
   - Review order details and customer information
   - Verify payment status

2. **Order Fulfillment**
   - Update order status to "Processing"
   - Pick and pack items
   - Generate shipping labels
   - Update status to "Shipped"

3. **Order Completion**
   - Customer receives order
   - Update status to "Completed"
   - Process any returns or exchanges

### Order Status Management

Available order statuses:
- **Pending**: Awaiting payment confirmation
- **Processing**: Payment confirmed, preparing for shipment
- **Shipped**: Order has been dispatched
- **Delivered**: Order received by customer
- **Cancelled**: Order has been cancelled
- **Refunded**: Order has been refunded

### Order Actions

- **View Order Details**: Complete order information
- **Print Invoice**: Generate printable invoice
- **Print Shipping Label**: Create shipping labels
- **Send Notification**: Email customer about order status
- **Process Refund**: Handle returns and refunds

## Frequently Asked Questions

**Q: How do I reset my password?**
A: Click "Forgot Password" on the login page and follow the email instructions.

**Q: Can I import products from a CSV file?**
A: Yes, use the "Import Products" feature in the Products section.

**Q: How do I set up automatic tax calculation?**
A: Configure tax rules in Settings → Tax Configuration.

**Q: Can I customize the order notification emails?**
A: Yes, email templates can be customized in Settings → Email Templates.

**Q: How do I track my inventory across multiple locations?**
A: Enable multi-location inventory in Settings → Inventory Configuration.
